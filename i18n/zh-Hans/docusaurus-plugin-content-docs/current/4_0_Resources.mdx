# 产品资源下载

欢迎来到 SENSING 资源中心。您可以通过以下方式轻松访问和下载所需的技术资源：

- **按产品型号搜索**：获取 **产品规格书**、**3D模型**、**SerDes初始化脚本** 和 **图像样本**。
- **按序列号（SN）搜索**：获取您特定设备独有的 **内参标定文件**。

- **产品型号** (例如 `SG3-ISX031C-GMSL2`): 标识类型和规格的特定摄像头名称。通常印在 **外包装** 或 **顶部标签** 上。
- **序列号/SN码** (例如 `H190S-E01220525`): 您的特定摄像头单元的唯一标识符。通常位于设备的 **底部标签** 上。
